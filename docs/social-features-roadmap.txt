SPRINT 7: Challenges, Sharing & Accountability
----------------------------------------------
Status: COMPLETED

Features Built:

7A: Challenges System
- Time-limited competitions between friends
- Challenge types: fitness (workouts/volume/XP), travel (locations/cities), today (habits/streak)
- Create/join/leave challenges with friend invites
- Real-time leaderboard with automatic score calculation
- XP rewards for winners
- Challenge status management (draft/active/completed)

7B: Activity Sharing
- Shareable cards for achievements, workouts, challenges
- Social sharing (Twitter, WhatsApp)
- Copy link and download image functionality
- RPG-themed share card design

7C: Accountability Partners
- Request accountability partnerships with friends
- Set shared goals with progress tracking
- Daily check-ins with mood tracking
- Nudge notifications (limit 1 per day)
- Partnership streak calculation

Database Schema:
- challenges (id, creator_id, title, type, app_id, metric, target, start_date, end_date, status, xp_reward, icon)
- challenge_participants (challenge_id, user_id, status, score, rank, joined_at)
- accountability_partnerships (id, requester_id, partner_id, status, message, started_at)
- partnership_goals (partnership_id, type, title, target, current_user1, current_user2, period)
- partnership_checkins (partnership_id, user_id, date, completed, notes, mood)
- partnership_nudges (partnership_id, sender_id, recipient_id, message, read)

API Endpoints:
Challenges:
- GET/POST /api/challenges
- GET/PUT/DELETE /api/challenges/[id]
- POST /api/challenges/[id]/join
- POST /api/challenges/[id]/leave
- GET /api/challenges/[id]/leaderboard

Accountability:
- GET/POST /api/accountability
- POST /api/accountability/[id]/respond
- GET/POST/PUT/DELETE /api/accountability/[id]/goals
- GET/POST /api/accountability/[id]/checkin
- GET/POST /api/accountability/[id]/nudge

Sharing:
- GET /api/share/achievement/[id]
- GET /api/share/workout/[id]
- GET /api/share/challenge/[id]

Components:
- components/challenges/ChallengeCard.tsx
- components/challenges/ChallengeCreator.tsx
- components/challenges/ActiveChallenges.tsx
- components/accountability/AccountabilityPartnerCard.tsx
- components/accountability/AccountabilityPartners.tsx
- components/share/ShareButton.tsx
- components/share/ShareModal.tsx
